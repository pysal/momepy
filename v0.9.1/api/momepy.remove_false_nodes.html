momepy.remove_false_nodes#

momepy.remove_false_nodes(gdf)[source]#

Clean topology of existing LineString geometry by removal of nodes of degree 2. Returns the original gdf if there’s no node of degree 2. Some geometries may be forced to 2D where a Z coordinate is present.

Parameters:
gdfGeoDataFrame, GeoSeries, array of shapely geometries

(Multi)LineString data of street network

Returns:
gdfGeoDataFrame, GeoSeries
