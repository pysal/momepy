# Network Analysis

Part of morphometric analysis is the analysis of street networks. While
`momepy` currently focuses more on smaller-scale analysis, some network-based
characters are included. The main difference here is that functions in the
`graph` module, allowing this kind of study, are based on `networkx.Graph`,
not `geopandas.GeoDataFrame`.
